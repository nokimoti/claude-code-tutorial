用語集

API キー (API Key)

API を使うときに必要な、本人確認のための 合鍵 (パスワードのような長い文字列)。 Anthropic のサイトから発行できます。

API とは?

API (Application Programming Interface) は、 「ソフトウェアどうしが連絡を取り合うための窓口」のことです。 Claude Code のように Claude と通信する道具は、すべてこの API を経由しています。

料金が発生するので大切に扱う

API キーは 使った分だけ料金が発生 します (従量課金)。 キーが漏れると、他人があなたのアカウントで Claude を使い放題になってしまうので、 取り扱いには注意が必要です。

  • Git にコミットしない (CLAUDE.md やスクリプトに直接書かない)
  • SNS やスクショに写り込まないように気をつける
  • 万が一漏れたら、Anthropic コンソールで「無効化 (revoke)」して再発行

個人なら Pro/Max プランから始めるのが安心

個人で使い始めるなら、API キーを使わずに月額固定の Claude Pro / Max プランを選ぶほうがシンプルです。 そのほうが「使いすぎて高額請求」の心配もありません。
