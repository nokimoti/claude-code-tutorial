用語集

IDE (統合開発環境)

コードエディタに、ファイラー・ターミナル・デバッガなどを統合したアプリのこと。 Integrated Development Environment の略です。

代表的な IDE

  • VS Code (Visual Studio Code) — Microsoft 製、無料、もっとも広く使われている
  • JetBrains 系 — IntelliJ IDEA、PyCharm、WebStorm など、言語ごとに最適化
  • Cursor — VS Code をベースに AI 機能を強化したエディタ

Claude Code との連携

VS Code と JetBrains 系には Claude Code 用の拡張機能が用意されています。 入れると、エディタ内に Claude Code のサイドパネルが開き、 開いているファイルが自動でコンテキストに含まれるなどの恩恵があります。
