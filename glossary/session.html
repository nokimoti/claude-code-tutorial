セッション (Session)

Claude Code を起動してから終了するまでの、ひとかたまりの対話 のこと。 履歴・コンテキスト・モード設定はセッション単位で管理されます。

セッションは保存されている

/exit や Ctrl+C でセッションを終了しても、内容はパソコンに記録されています。 claude -c で直前のセッションを、claude --resume で過去のセッションから選んで再開できます。

セッションを切る・つなげるの使い分け

状況やること
同じ流れの続きそのまま続ける
同じプロジェクトで話題が変わる/clear でリセット
会話が長すぎる/compact で要約圧縮
翌日に持ち越す/exit → 翌日 claude -c
