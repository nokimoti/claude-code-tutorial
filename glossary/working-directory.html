ワーキングディレクトリ (Working Directory)

ターミナル でいま「自分がいる」フォルダのこと。 カレントディレクトリ (current directory) とも呼ばれます。

「いる場所」のイメージ

ターミナルでは常に「あなたはどこかのフォルダの中にいる」状態になっています。 その「どこ」がワーキングディレクトリです。 pwd (Linux/macOS) または pwd / Get-Location (PowerShell) で現在地を確認できます。

cd コマンドで別のフォルダに移動できます。 

cd C:\Users\you\projects\my-app

Claude Code との関係

Claude Code は 起動したときのワーキングディレクトリを「作業対象のプロジェクト」として認識 します。 なので「このプロジェクト」「このフォルダ」と言ったときは、ワーキングディレクトリのことを指します。
