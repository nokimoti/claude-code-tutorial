Claude Desktop アプリ

Anthropic が提供する、Windows / macOS / Linux 向けの デスクトップアプリ。 ウィンドウから日本語で話しかけるだけで Claude Code を使えます。

このサイトで使うアプリ

Claude Code は本来「コマンドラインで動くツール」ですが、 Claude Desktop アプリを使うと すべてマウスとキーボードで完結 します。 ターミナルや CLI の知識がなくても始められるのが大きな利点です。

主な機能

  • 会話形式の入力欄と返答ストリーム表示
  • サイドバーから過去の会話を一覧・再開
  • ファイル編集前のサイドバイサイド差分プレビュー
  • ボタンクリックで 許可モード を切替
  • ドラッグ&ドロップでファイル添付
  • 設定画面 (歯車アイコン) からモデル・許可・コネクターを GUI で管理

入手方法

公式サイト claude.ai/download から 自分の OS 用のインストーラーをダウンロードします。

ブラウザ版 (claude.ai) との違い

  • ブラウザ版: 会話・文章生成は得意。ファイル操作はアップロードベース、コマンド実行は不可
  • Desktop アプリ: パソコン内のファイルを直接読み書き・コマンド実行ができる (許可制)
