スラッシュコマンド (Slash Command)

入力欄で / から始めると使える、 Claude ではなく Claude Code 自体に対する命令。

覚えておきたい主なコマンド

  • /help — 使えるコマンド一覧
  • /clear — 会話履歴をリセット
  • /init — CLAUDE.md を自動生成
  • /exit — セッション終了
  • /permissions — 許可ルールの確認
  • /model — AIモデル の切り替え
  • /compact — 会話の圧縮
  • /agents — サブエージェント の管理
  • /mcp — MCP サーバーの管理

カスタムスラッシュコマンドも作れる

.claude/commands/ フォルダに Markdown ファイルを置くと、 自分専用のスラッシュコマンドとして使えます。 たとえば commands/test.md に「テストを実行して結果を要約して」と書いておけば、 入力欄で /test と打つだけでその指示が送れます。
