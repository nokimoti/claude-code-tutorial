用語集

フック (Hook)

ある操作の 前後 に、決まったコマンドを 自動で走らせる仕組み。 settings.json に書きます。

いつ走らせるか

  • PreToolUse — ツールが使われる直前 (許可前にチェックを挟むなど)
  • PostToolUse — ツールが使われた直後 (編集後に linter を走らせるなど)
  • UserPromptSubmit — ユーザーが送信した直後
  • Stop — Claude が応答を終えたとき
  • SessionStart / SessionEnd — セッション開始時・終了時

例: 編集後に自動でフォーマット

{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Edit|Write",
        "hooks": [
          { "type": "command", "command": "npm run format" }
        ]
      }
    ]
  }
}

この設定があると、Claude がファイルを編集するたびに自動で npm run format が走ります。

禁止用にも使える

PreToolUse で「特定の条件のツール呼び出しをブロック」することもできます。 たとえば「.env ファイルへの書き込みは絶対禁止」のようなルールを書けます。
