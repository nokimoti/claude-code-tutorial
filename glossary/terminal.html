用語集

ターミナル (Terminal)

文字でパソコンを操作するための窓口となるアプリ。黒い画面に $ や > が出て、そこに文字を打ち込んで命令します。
Claude Desktop アプリ を使う限り、ターミナルを自分で開く必要はありません。 コマンドの実行はアプリの中で完結します。 ターミナルは「Claude Code の CLI 版 を使う場合や、開発者が普段の作業で使うもの」と覚えておけば十分です。

OS によって名前が違う

  • Windows: PowerShell、Windows Terminal、コマンドプロンプト
  • macOS: ターミナル.app (Terminal.app)
  • Linux: GNOME Terminal、Konsole など

呼び方は違っても、やれることはどれもほぼ同じです。

「ターミナル」と「シェル」の関係

ターミナルは「窓口アプリ」、シェルは「窓口の中で実際にコマンドを解釈する係」です。 たとえば Windows Terminal の中で PowerShell が動く、というように 1 セットで使われます。
