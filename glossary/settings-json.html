用語集

settings.json

Claude Code の 設定ファイル。 許可ルール、フック、環境変数、モデル選択などを JSON 形式で書いておけます。

3 つの設定ファイルの優先順位

  1. .claude/settings.local.json — プロジェクト・自分専用 (Git に入れない)
  2. .claude/settings.json — プロジェクト・チーム共有
  3. ~/.claude/settings.json — パソコン全体・自分専用

上にあるほど優先されます。

書ける主な項目

{
  "permissions": {
    "allow": ["Bash(git status)", "Read(*)"],
    "deny":  ["Bash(rm -rf:*)"]
  },
  "hooks": {
    "PostToolUse": [...]
  },
  "env": { "DEBUG": "true" },
  "model": "claude-sonnet-4-6"
}
