用語集

ツール (Tool)

Claude が外の世界に働きかけるための 手足にあたる機能。 ファイルを読む・書く・コマンドを実行する、などはすべてツール経由で行われます。

主なツール

ツール名役割
Readファイルの中身を読む
Editファイルの一部を書き換える
Writeファイルを新規作成・上書き
Bashシェルのコマンドを実行
Globファイル名のパターン検索
Grepファイルの中身を検索
WebFetch / WebSearchWeb の情報を取ってくる
TodoWriteタスクリストを管理

ツールが使われると画面に出る

Claude がツールを使うと、ターミナルにこんな表示が出ます。 

● Read(memo.txt)
  ⎿  Read 4 lines

● が「ツールを使った」マーク、その横にツール名と引数、 ⎿ 以下に結果が表示されます。

ツールごとに 許可 の対象になる

Edit や Bash のように外に影響を与えるツールは、 実行前に必ず確認画面が出るようになっています。 Read のような読み取り専用のツールは、初期状態で自動許可です。
