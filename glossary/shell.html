用語集

シェル (Shell)

入力されたコマンドを 解釈して実行する 役割のプログラム。 Claude Desktop アプリには、これがあらかじめ組み込まれています。

アプリの中で勝手に動いている

Claude が「コマンドを実行する」ときは、アプリ内蔵のシェルにコマンドを渡して動かしています。 あなたが ターミナル を開く必要はなく、すべてアプリの中で完結します。

主なシェルの種類

  • bash — 古くから使われる定番。Linux サーバーなどで標準
  • zsh — bash の後継的存在。macOS の標準シェル
  • PowerShell — Windows 標準。文法が独特
  • fish — 補完がリッチで使いやすい

シェルが違うとコマンドの書き方が少し違います。 たとえば「環境変数を見る」だけでも bash なら echo $PATH、 PowerShell なら $env:PATH のように書き方が変わります。

Claude Code はあなたの OS に合わせて、適切なシェル用のコマンドを自動で組み立ててくれます。
