Anthropic

AI アシスタント Claude を開発・提供している会社。本社はアメリカ・サンフランシスコ。

提供しているサービス

  • claude.ai — ブラウザで使える対話型 AI
  • Claude Code — このサイトで扱っているターミナル向けツール
  • Anthropic API — 自分のアプリに Claude を組み込むための API

安全で役に立つ AI を作ることを目標に掲げており、Claude シリーズはそのコア製品です。
