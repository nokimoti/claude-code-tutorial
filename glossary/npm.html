用語集

npm (Node Package Manager)

Node.js 用の パッケージ管理ツール。 ライブラリやコマンドラインツールを 1 行のコマンドでインストールできます。

世界中のパッケージが集まる場所

npm には世界中の開発者が公開した何百万ものパッケージ (= プログラムの部品) が登録されています。 Claude Code もその中のひとつ @anthropic-ai/claude-code として配布されています。

よく使うコマンド

  • npm install -g (パッケージ名) — グローバル(パソコン全体)にインストール
  • npm install (パッケージ名) — 現在のプロジェクトにだけインストール
  • npm update -g (パッケージ名) — 最新版に更新
  • npm uninstall -g (パッケージ名) — アンインストール

-g は global の略で、これを付けると「どのフォルダにいても使えるコマンドとして入れる」になります。
