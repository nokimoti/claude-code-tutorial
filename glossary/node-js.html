用語集

Node.js

JavaScript で書かれたプログラムを パソコン上で動かすための土台 (ランタイム)。
Claude Desktop アプリ を使うだけなら、Node.js を別途インストールする必要はありません。 Node.js が必要になるのは、Claude Code の CLI 版 を使う場合や、 Node.js 製のプロジェクトを Claude にいじってもらう場合だけです。

JavaScript はブラウザだけじゃない

もともと JavaScript は Web ブラウザの中で動く言語でしたが、 Node.js のおかげで、サーバーやコマンドラインツールも JavaScript で作れるようになりました。 Claude Code の CLI 版もこの Node.js で動いています。

必要になる場面

  • Claude Code の CLI 版 (claude コマンド) を使うとき → Node.js 18 以上が必要
  • Node.js 製のプロジェクト (Express、Next.js など) を Claude に編集させるとき → 動作確認のため

公式サイトの LTS 版 (長期サポート版) を入れておくのが無難です。

  • 公式: nodejs.org

入っているかは、ターミナル で node --version を実行すれば確認できます。
