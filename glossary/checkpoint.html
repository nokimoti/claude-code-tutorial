用語集

チェックポイント (Checkpoint)

Claude Code が会話の進行中に作る 「巻き戻しポイント」。 過去のあるメッセージまで戻って、そこから別の指示を出し直せます。

使い方

入力欄で Esc を 2 回押すと、過去のメッセージ一覧が表示されます。 戻りたい時点を選ぶと、それ以降のやり取りはキャンセルされ、 その地点から会話を再開できます。

ファイルの状態も、その時点のものに巻き戻ります (Claude が編集したぶんに限る)。 ゲームのセーブ&ロードのような感覚です。

使いどころ

  • 方針を間違えて延々と修正させてしまった → 最初の指示の地点まで戻して聞き方を変える
  • 大量に変更した結果が気に入らない → 1 ターン前に戻す

Git の commit との違い

チェックポイントは「会話と編集のセット」を巻き戻します。 Git は「ファイルの状態だけ」を巻き戻します。 Git でこまめに commit していれば、Claude Code に頼らずファイル単位で戻せます。両方併用するのがおすすめです。
