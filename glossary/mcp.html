MCP (Model Context Protocol)

Claude のような AI に 外部システムへの手足を生やす ための共通規格。 MCP に対応したサーバーを接続することで、Claude が新しい ツール を使えるようになります。

たとえばこんな連携ができる

  • Notion / Slack / GitHub の中身を読み書きする
  • 社内のデータベースに直接クエリを投げる
  • ブラウザを操作して Web スクレイピング
  • Figma や Google Drive のファイルを開く

公式・非公式を含めて、世界中で MCP サーバーが公開されており、 組み合わせ次第で Claude Code がぐっと強力になります。

接続方法

.claude/mcp.json や .mcp.json に接続先を書き、 入力欄で /mcp と打つと管理画面が開きます。 接続したサーバーが提供するツールは、自動で Claude が使えるようになります。

「規格」だから何にでもつなげる

MCP は Anthropic が公開している規格で、 Claude 以外の AI ツールも MCP に対応しはじめています。 ある場所で作った MCP サーバーが、別の AI でもそのまま使える、というのが狙いです。
