許可モード (Permission Mode)

Claude Code が ツール を使うときに、 どこまで自動で許可するかを切り替える設定。 4 段階あります。

4 つのモード

  • Default — 編集とコマンド実行のたびに確認 (初期状態)
  • Plan — 調査・読み取りだけ。書き換えは一切できない
  • Accept Edits — ファイル編集は自動許可、コマンド実行は確認
  • Bypass — すべて確認なし (危険)

入力欄で Shift + Tab を押すたびにモードが切り替わります。

Bypass モードの注意

Bypass モードはすべての操作を確認なしで実行します。 使い捨てのコンテナや VM 内に限定し、本番環境やメインの作業フォルダでは使わないでください。

細かいルールは settings.json で

モード一括ではなく「git status はいつも許可」のような細かい設定をしたい場合は、 .claude/settings.json に書きます。
