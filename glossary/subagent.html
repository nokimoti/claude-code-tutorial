サブエージェント (Sub-agent)

特定の役割や得意分野を持った Claude の分身。 メインの Claude が必要に応じて呼び出して、一部の仕事を任せます。

役割を分担できる

たとえば次のようなサブエージェントを作っておけます。

  • code-reviewer — コードを読み、改善点を出すだけのレビュー専門
  • test-runner — テストを実行して結果を整理する役
  • doc-writer — README やコメントの整備担当

メインの Claude が「ここはレビュー専門に任せよう」と判断すると、 その役割のサブエージェントを呼び出します。サブエージェントには独立した コンテキストウィンドウ が与えられるので、 余計な履歴に邪魔されずに集中して仕事ができます。

定義場所

.claude/agents/ フォルダに Markdown ファイルとして書きます。 入力欄で /agents と打つと、新規作成や編集の UI が開きます。
