AIモデル

文章を理解して返答を生成する AI 本体 のこと。 Claude Code の場合は Anthropic の Claude シリーズが使われます。

Claude シリーズの主なモデル

  • Opus — 一番賢く、複雑な作業向き。少し料金が高め
  • Sonnet — バランス型。日常使いの主力
  • Haiku — 軽量で高速、安価。短い作業向き

用途に合わせて入力欄で /model と打つと切り替えられます。 普段は Sonnet、難しい設計のときだけ Opus、というような使い分けが一般的です。

「考えている」ように見えるしくみ

モデルは膨大な文章でトレーニングされており、入力を受け取ると 「次に来るのに最もふさわしい言葉」を 1 つずつ予測しながら文章を組み立てます。 実際に「考えて」いるかどうかは哲学的な議論ですが、 使う側からすれば賢いアシスタントとして振る舞う、と考えてかまいません。
