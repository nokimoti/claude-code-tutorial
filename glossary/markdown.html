用語集

Markdown

# で見出し、- で箇条書き、というように、 記号で簡単に書式を表現できる 軽量なテキスト形式。

覚えやすい記法

  • # 見出し1 / ## 見出し2
  • - りんご で箇条書き
  • **太字** や *斜体*
  • バッククォートで `code`
  • 3 連バッククォートでコードブロック

拡張子は .md。GitHub の README、ドキュメント、ブログ記事など、 開発まわりの文章はほぼ Markdown で書かれます。

Claude Code での使われどころ

  • CLAUDE.md — プロジェクトの指示書
  • カスタム スラッシュコマンド の定義ファイル
  • スキル の説明文

Claude も Markdown 形式で答えるので、ターミナルでも見出しやリストが見やすく整形されます。
