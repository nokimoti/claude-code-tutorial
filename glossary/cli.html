用語集

CLI (Command Line Interface) / コマンドライン版

キーボードから 文字 (コマンド) を入力して操作する、ソフトウェアの操作方式。 アイコンをクリックして使う GUI の対義語です。
このサイトでは Claude Desktop アプリ を使うので、 CLI を直接使う場面はありません。「そういうものもあるんだな」程度の理解で大丈夫です。

もう少し詳しく

普段使っているアプリの多くは、ウィンドウやアイコンをマウスでクリックして操作します。 これを GUI (Graphical User Interface) といいます。

これに対して、ターミナル に文字を打って指示する操作方式を CLI と呼びます。 慣れるまでは敷居が高く感じますが、大量の処理を一気に動かしたり、自動化したりするのが得意です。

Claude Code には CLI 版もある

Claude Code はもともとコマンドラインツールとして誕生しました。 開発者向けに claude コマンドで起動する CLI 版 が今でも提供されています。

  • インストール: npm 経由 (npm install -g @anthropic-ai/claude-code)
  • 動作環境: Node.js 18 以上が必要
  • 使い方: ターミナル で claude と打って起動

機能はほぼ Desktop 版と同じです。コマンドラインに慣れた方や、サーバーで自動化したい場合に便利です。
