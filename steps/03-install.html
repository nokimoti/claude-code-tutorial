STEP 03

インストール

このステップのゴール

  • Claude Desktop アプリ をパソコンにインストールする
  • アプリのアイコンをクリックして起動できる状態にする

1. ダウンロードページを開く

ブラウザで claude.ai/download を開きます。 お使いの OS が自動で判別され、ダウンロード用のボタンが表示されます。

  • Windows: 「Download for Windows」→ .exe ファイルがダウンロードされます
  • macOS: 「Download for Mac」→ .dmg ファイルがダウンロードされます

2. インストーラーを実行する

Windows の場合

  1. ダウンロードした Claude-Setup.exe をダブルクリック
  2. 「ユーザーアカウント制御」が出たら「はい」
  3. インストーラーの画面に従って「次へ」を進める
  4. 完了するとデスクトップとスタートメニューに Claude アイコンが追加されます

macOS の場合

  1. ダウンロードした Claude.dmg をダブルクリックして開く
  2. 表示されたウィンドウの中で、Claude のアイコンを「Applications」フォルダにドラッグ
  3. 「Applications」フォルダから Claude をダブルクリックで起動
  4. 初回起動時に「ダウンロードされたアプリです。開いてもよいですか?」と聞かれるので「開く」
Mac でアプリが「壊れているため開けません」と表示された場合は、 システム設定 → プライバシーとセキュリティ で「このまま開く」を選んでください。

3. インストールできたか確認

アプリ一覧 (Windows のスタートメニュー、macOS の Launchpad / Applications フォルダ) に Claude のアイコンが見つかれば成功です。

クリックして起動すると、ログイン画面が現れます。次の STEP 04 で初回起動を扱います。

うまくいかないときは

ダウンロードボタンが見つからない

ブラウザの言語設定が日本語の場合、「ダウンロード」ボタンになっていることがあります。 公式トップ claude.ai 右下のリンクから辿ることもできます。

セキュリティソフトに止められた

会社支給のパソコンなどでは、セキュリティソフトがインストーラーをブロックすることがあります。 情報システム部門に相談してください。

必ず公式サイトからダウンロードしてください。 検索結果で似た名前の偽サイトが上位に出ることがあります。 URL が claude.ai または anthropic.com であることを確認しましょう。

アップデートのしかた

Claude Desktop アプリは、新しいバージョンが出ると自動でお知らせしてくれます。 画面に「アップデートがあります」と表示されたら、案内に従ってボタンをクリックするだけです。 手動でやることはほとんどありません。

このステップのまとめ

  • 公式サイト claude.ai/download からインストーラーをダウンロード
  • 普通のアプリと同じ手順でインストール
  • アプリ一覧に Claude が現れれば完了
