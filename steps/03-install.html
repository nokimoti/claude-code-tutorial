STEP 03

インストール

このステップのゴール

  • Claude Code を自分のパソコンにインストールする
  • インストールが成功したことを確認する

1コマンドで入ります

Claude Code のインストールは、ターミナル で次のコマンドを 1 行打つだけです。 

npm install -g @anthropic-ai/claude-code

npm は npm といって、 Node.js 用のパッケージ管理ツールです。 -g は「グローバル(=パソコン全体)にインストールする」という意味で、 これを付けるとどのフォルダからでも claude コマンドが使えるようになります。

ダウンロードに数分かかります。途中で止まったように見えてもキャンセルせず待ちましょう。

うまく入ったか確認する

インストールが終わったら、ターミナルで次のコマンドを打ってみます。 

claude --version

1.x.x (Claude Code) のようなバージョンが表示されれば成功です。

うまくいかないときは

「command not found: npm」と出た

Node.js がまだ入っていない可能性が高いです。 STEP 02 に戻って Node.js をインストールしてから、もう一度試してください。

「Permission denied」「EACCES」と出た (macOS / Linux)

パソコンの設定によっては、グローバルインストールに管理者権限が必要なことがあります。 その場合は次のコマンドで sudo を付けて実行します。 

sudo npm install -g @anthropic-ai/claude-code

「claude: command not found」と出る (インストール直後)

ターミナルが古い情報のままになっていることがあります。 ターミナルを 一度閉じて開き直して から、もう一度 claude --version を試してください。

Windows の場合、稀に PowerShell の「実行ポリシー」が厳しく設定されていて claude が起動できないことがあります。 その場合は管理者として PowerShell を開き、 Set-ExecutionPolicy -Scope CurrentUser RemoteSigned を実行してください。

アップデートのしかた (将来の参考に)

Claude Code は頻繁に新機能が追加されます。最新版に上げるには、 同じく npm で次のコマンドを打ちます。 

npm update -g @anthropic-ai/claude-code

また、Claude Code 自身が新しいバージョンを検知すると、起動時に「アップデートがあります」と教えてくれます。

このステップのまとめ

  • インストールは npm install -g @anthropic-ai/claude-code の 1 行で完結
  • 確認は claude --version
  • うまくいかないときは Node.js が入っているか・ターミナルを再起動したかを確認
