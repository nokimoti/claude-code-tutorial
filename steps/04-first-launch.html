STEP 04

はじめての起動

このステップのゴール

  • 練習用のフォルダを作って Claude Code を起動できる
  • 初回ログインを完了して、プロンプトの入力欄が表示されるところまでたどり着く

1. 練習用のフォルダを作る

Claude Code は 「いま自分がいるフォルダ」を作業対象にする 道具です。 この「いるフォルダ」のことを ワーキングディレクトリ といいます。

まずは練習用に空のフォルダを作りましょう。デスクトップやドキュメントの中など、好きな場所で構いません。 

mkdir claude-test
cd claude-test

mkdir はフォルダを作るコマンド、cd はそのフォルダの中に移動するコマンドです。

2. claude と打つだけ

その状態で次のコマンドを打ちます。 

claude

これだけで Claude Code が起動します。初回起動時は、ターミナルにカラフルなロゴと 「Welcome」のメッセージが出るはずです。

3. 初回はログインが必要

最初の 1 回だけ、Anthropic アカウントとの紐付けが必要です。 画面の指示に従って、次のどちらかを選びます。

  • Claude アカウントでログイン — Pro / Max プランの方はこちら。ブラウザが開いて、claude.ai のログイン画面に飛びます。
  • API キー を入力 — 従量課金で使う方はこちら。

ブラウザでログインすると、自動でターミナルに戻り「ログインに成功しました」と表示されます。

4. 入力欄が出ればOK

ログインが終わると、こんな感じの入力欄が画面の下に表示されます。 

> Try "How does this codebase work?"
  ────────────────────────────────────

ここにキーボードから日本語や英語を入力すると、Claude Code に伝わります。 これで セッション がスタートしました。

終了するには

終了したいときは、入力欄に /exit と打つか、Ctrl + C を 2 回押します。 これで元のターミナルに戻ります。

終了しても会話の内容は記録されています。STEP 13 で「続きから再開する」やり方を学びます。

このステップのまとめ

  • 作業したいフォルダに cd で移動する
  • claude と打つだけで起動する
  • 初回はログイン or API キーを聞かれる
  • 入力欄が出たら準備完了 — 次は実際に話しかけます
