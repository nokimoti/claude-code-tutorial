STEP 04

はじめての起動

このステップのゴール

  • Claude Desktop アプリを起動してログインを完了する
  • 練習用のフォルダを「作業対象」としてアプリに認識させる

1. アプリを起動する

アプリ一覧から Claude のアイコンをクリックします。

  • Windows: スタートメニュー or デスクトップから
  • macOS: Launchpad、Applications フォルダ、Spotlight (⌘ + Space) で「Claude」と検索

2. 初回はログインが必要

初回起動時、ログイン画面が出ます。次のどちらかを選びます。

  • Claude アカウントでログイン — Pro / Max プランの方はこちら。「Sign in」を押すと既定のブラウザが開いて認証されます。
  • API キー を入力 — 従量課金で使う方はこちら。設定画面から API キーを貼り付けます。

ブラウザでの認証が終わると、自動でアプリに戻ってきて、メイン画面が表示されます。

3. 練習用のフォルダを作る

Claude Code は 「ひとつのフォルダを作業対象として」働く 道具です。 学習用に空のフォルダをひとつ作っておきましょう。

  1. デスクトップでもドキュメントでも、好きな場所で右クリック → 「新しいフォルダー」
  2. フォルダ名は claude-test にしておく

4. アプリにフォルダを開かせる

アプリのメイン画面で「フォルダを開く」(Open Folder) をクリックします。 ファイル選択ダイアログが出るので、いま作った claude-test フォルダを選んで「開く」を押します。

これで Claude が そのフォルダの中にあるファイルを読み書きできる状態 になりました。 この「いま開いているフォルダ」のことを プロジェクトフォルダ と呼びます。

別のフォルダで作業したくなったら、いつでもメニューから「フォルダを切り替える」で別の場所を開けます。

5. 入力欄が出れば準備完了

ウィンドウの下のほうに、メッセージを書き込む入力欄があります。 

┌──────────────────────────────────────┐
│ Claude に話しかけてみましょう…       │
└──────────────────────────────────────┘

ここにキーボードから日本語や英語を入力すると、Claude に伝わります。 これで セッション がスタートしました。

終了するには

普通のアプリと同じです。ウィンドウ右上の「×」(macOS は左上の赤丸) を押す、 もしくは Ctrl + Q (macOS は ⌘ + Q) で終了します。

終了しても会話の内容はアプリに保存されています。STEP 13 で「続きから再開する」やり方を学びます。

このステップのまとめ

  • アプリのアイコンをクリックして起動
  • 初回はログイン or API キーを聞かれる
  • 「フォルダを開く」で作業対象のプロジェクトフォルダを指定
  • 入力欄が出たら準備完了 — 次は実際に話しかけます
