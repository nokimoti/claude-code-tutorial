STEP 01

Claude Codeとは何か

このステップのゴール

  • Claude Code が「何をしてくれる道具なのか」をイメージできる
  • ブラウザ版の Claude (claude.ai) との違いを理解する

Claude Code は「ターミナルで動く AI ペアプログラマ」

Claude Code は、Anthropic 社が提供する、 ターミナル 上で動く AI コーディングアシスタントです。 自然な日本語(または英語)で指示するだけで、あなたのパソコンの中で コードを読んだり、書いたり、コマンドを実行したりしてくれます。

たとえば、こんな経験はありませんか?

  • 「このバグの原因がどこにあるのか、ファイルを開いて探すのが大変」
  • 「同じような修正を 20 ファイルに繰り返したくない」
  • 「テストの書き方がわからないから、まず雛形だけ作ってほしい」
  • 「動かないエラーメッセージをそのまま渡して、原因を一緒に考えてほしい」

Claude Code は、こうした作業を「お願いベース」で進められる道具です。

具体的に何ができる?

  1. ファイルを読む — 「このプロジェクトの構造を教えて」と聞ける
  2. ファイルを編集する — 「この関数のバグを直して」とお願いできる
  3. コマンドを実行する — 「テストを走らせて結果を見て」と頼める
  4. 調べてくれる — Web から情報を取ってきてもらえる
  5. 計画を立ててくれる — いきなり書き始める前に、手順を提示してくれる

ブラウザ版 Claude との違い

ブラウザの claude.ai でも会話できますが、 Claude Code には大きな違いがあります。

項目claude.ai (ブラウザ)Claude Code (ターミナル)
動く場所ブラウザの中あなたのパソコンの中
ファイル操作アップロードして読ませる直接読み書きできる
コマンド実行できないできる (許可制)
得意なこと会話・文章生成コード作業・自動化

「CLI」と聞いて怖がらなくて大丈夫

Claude Code は CLI ツール なので、 操作はすべて ターミナル から行います。 でも、難しいコマンドを覚える必要はありません。 起動したあとは、人と話すように 日本語で指示するだけ です。

Claude Code 自体が「あなたの代わりにコマンドを打つ存在」なので、 むしろあなたはコマンドを覚えなくてよくなる、という逆転現象が起きます。

このステップのまとめ

  • Claude Code は Anthropic 製の AI コーディングアシスタント
  • ターミナル から起動する CLI ツール
  • ファイルを読む・書く・コマンドを実行することができる
  • 操作は基本的に「日本語で話しかけるだけ」で OK
