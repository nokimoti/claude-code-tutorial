STEP 02

用意するもの

このステップのゴール

  • Claude Code を使うために必要な3つのものを把握する
  • Node.js がパソコンに入っているかを確認できる

必要なものは大きく3つ

  1. ターミナル — お使いの OS に最初から入っています
  2. Node.js — Claude Code 本体を動かすために必要
  3. Claude のアカウント — Anthropic のサービスを使う権利

1. ターミナル

ターミナル は、文字でパソコンを操作するための窓口です。 OS によって名前が違いますが、どの OS にも最初から入っています。

  • Windows: PowerShell または Windows Terminal
  • macOS: ターミナル (Terminal.app)
  • Linux: GNOME Terminal、Konsole など

まずは「ターミナル」というアプリを見つけて、起動できることを確認しておきましょう。 起動するとこんな画面が開きます。 

PS C:\Users\you> _

この > や $ といった記号のあとに文字を入力できる場所のことを プロンプト と呼びます (混乱しやすいので注意: AI に渡す指示文のことも プロンプト と呼びます)。

2. Node.js

Node.js は、JavaScript で書かれたプログラムをパソコン上で動かす仕組みです。 Claude Code は Node.js で動いているので、これが先に必要になります。

入っているかは、ターミナルで次のコマンドを実行すれば確認できます。 

node --version

v20.x.x のようにバージョン番号が出ればインストール済みです。 command not found や「コマンドではない」と出る場合は、まだ入っていません。 公式サイト nodejs.org から LTS 版をダウンロードしてインストールしてください。

Claude Code は Node.js 18 以上が必要です。古すぎるバージョンが入っている場合は最新の LTS 版に更新しましょう。

3. Claude のアカウント

Claude Code を使うには、課金プランの選択が必要です。 最初に選ぶのは次のどちらかです。

  • Claude Pro / Max プラン (月額のサブスク) — 個人で使うならこれが手軽です。 ブラウザの claude.ai で使うものと同じプランで、Claude Code もそのまま使えます。
  • API 利用 (従量課金) — Anthropic の API キー を取得して、使った分だけ支払う方式。 企業で使う場合や、組織のチームが共有する場合はこちらを選ぶことが多いです。

迷ったら Claude Pro プラン から始めるのがおすすめです。 アカウントは claude.ai から作成できます。

まだプランを契約していなくても大丈夫。STEP 04 で初回起動時にログインを求められるので、 そのときに案内に従って契約することもできます。

このステップのまとめ

  • 必要なのは「ターミナル」「Node.js」「Claude アカウント」の3つ
  • ターミナルは OS に最初から入っている
  • Node.js は node --version で入っているか確認できる
  • アカウントは Pro プラン or API キーのどちらか (個人なら Pro が手軽)
