STEP 02

用意するもの

このステップのゴール

  • Claude Code を始めるために必要な 2 つのものを把握する
  • 自分のパソコンが対応 OS かどうかを確認する

必要なものは大きく2つだけ

  1. 対応 OS のパソコン — Windows 10 以降 / macOS 12 以降
  2. Claude のアカウント — Anthropic のサービスを使う権利

この 2 つさえあれば始められます。プログラミング言語の知識や、特殊な開発環境のセットアップは不要です。

1. 対応 OS のパソコン

  • Windows: 10 / 11 (64bit)
  • macOS: 12 (Monterey) 以降
  • Linux: 多くのディストリビューションに対応 (アプリは AppImage / deb / rpm で配布)

自分のパソコンの OS バージョンの確認方法:

  • Windows: 「設定」→「システム」→「バージョン情報」
  • macOS: アップルメニュー→「この Mac について」

2. Claude のアカウント

Claude Code を使うには、課金プランの選択が必要です。 最初に選ぶのは次のどちらかです。

  • Claude Pro / Max プラン (月額のサブスク) — 個人で使うならこれが手軽です。 ブラウザの claude.ai で使うものと同じプランで、Claude Desktop アプリもそのまま使えます。
  • API 利用 (従量課金) — Anthropic の API キー を取得して、使った分だけ支払う方式。 企業で使う場合や、組織のチームが共有する場合はこちらを選ぶことが多いです。

迷ったら Claude Pro プラン から始めるのがおすすめです。 アカウントは claude.ai から作成できます。

まだプランを契約していなくても大丈夫。STEP 04 で初回起動時にログインを求められるので、 そのときに案内に従って契約することもできます。

あると役立つもの (必須ではない)

  • 練習用のフォルダ — テキストファイルを置いて遊ぶ、空のフォルダがあると進めやすいです。 STEP 04 で作ります。
  • VS Code などのエディタ — Claude が編集した結果を別画面で確認したいときに便利。なくてもアプリ内で完結します。

このステップのまとめ

  • 必要なのは「対応 OS のパソコン」と「Claude アカウント」の 2 つだけ
  • Windows 10+ / macOS 12+ ならそのまま使える
  • 個人なら Claude Pro プランから始めるのが手軽
