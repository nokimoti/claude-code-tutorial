このサイトについて

このサイトは Claude Code をまったく触ったことのない方 を対象に、 ひとつずつ小さなステップで学べるように作られています。 操作はすべて Claude Desktop アプリ から ── マウスとキーボードで完結します。コマンドを覚える必要はありません。 各ステップは 5〜10 分で読み終わるボリュームです。

文中の専門用語は こんなふうに リンクになっています。 意味がわからなくなったら、リンクをクリックすると用語の説明ページにジャンプできます。

このサイトで前提とする環境

  • OS: Windows 10 以降 または macOS 12 以降
  • アプリ: Claude Desktop アプリ (STEP 03 でインストールします)
  • アカウント: Claude Pro / Max プラン (STEP 02 で説明します)
プログラミング経験がなくても、ターミナル操作の知識がなくても大丈夫です。 アプリのウィンドウからすべて完結します。
