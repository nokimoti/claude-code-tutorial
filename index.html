このサイトについて

このサイトは、Claude Code をまったく触ったことのない方 を対象に、 ひとつずつ小さなステップで学べるように作られています。 各ステップは 5〜10分で読み終わるボリュームになっており、 手を動かしながら少しずつ進められる構成です。

文中の専門用語は こんなふうに リンクになっています。 意味がわからなくなったら、リンクをクリックすると用語の説明ページにジャンプできます。

このサイトで前提とする環境

  • OS: Windows / macOS / Linux いずれでも対応
  • ターミナル: Windows なら PowerShell、macOS なら Terminal、Linux なら標準のターミナル
  • 事前に Node.js のインストールが必要 (STEP 02 で説明します)
プログラミング経験がなくても大丈夫です。「こういう道具なんだな」というイメージをつかむことを優先しています。
